# Screenshots

The `.svg` files here are **UI mockups** that use the app's real palette and
layout, so the README has visuals without needing a device.

## Replace them with real screenshots

1. Run the app (`npx expo start` in `app/`) and open it in Expo Go or a simulator.
2. Capture each screen:
   - **Physical phone:** use the device screenshot shortcut.
   - **Android emulator:** the camera icon in the side toolbar.
   - **iOS simulator:** `Cmd+S` (saves to Desktop).
3. Save the images here, overwriting the mockups (keep the same names so the
   README picks them up automatically):
   - `01-auth.png` — sign in / demo
   - `02-dashboard.png` — daily dashboard
   - `03-analyze.png` — AI photo analysis result
   - `04-coach.png` — AI Meal Coach
4. In `README.MD`, change the `.svg` extensions to `.png` in the screenshots
   table (or just delete the mockup `.svg` files).
